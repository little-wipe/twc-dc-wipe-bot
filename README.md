## Usage

1. Копируем `config.example.js` в `config.js`
2. Указываем token
   аккаунта ([Инструкция](https://github.com/aiko-chan-ai/discord.js-selfbot-v13?tab=readme-ov-file#get-token-))
3. Указываем id каналов, в которых бот будет отвечать
4. `npm start`
